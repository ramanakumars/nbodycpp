\doxysubsection{Detailed Description}
Main rendering class for visualization. 

Handles SFML window creation, event processing, and rendering of particles, quadtree structure, and UI elements.

Features\+:
\begin{DoxyItemize}
\item Real-\/time particle visualization
\item \doxylink{classQuadTree}{Quad\+Tree} structure overlay
\item \doxylink{classParticle}{Particle} tracking (click to follow)
\item Zoom and pan controls
\item Velocity-\/based coloring (bound vs unbound orbits)
\item Pause/resume simulation 
\end{DoxyItemize}

Find particle at mouse cursor position. 

Find and track particle nearest to mouse cursor.

Uses \doxylink{classQuadTree}{Quad\+Tree} spatial query to find nearest particle to mouse click location. Updates track\+\_\+particle.

Algorithm\+:
\begin{DoxyEnumerate}
\item Convert mouse position from screen to simulation coordinates
\item Create search region (10\% of view width, centered on cursor)
\item Query \doxylink{classQuadTree}{Quad\+Tree} for particles in region
\item Find particle closest to cursor
\item Set as tracked particle
\end{DoxyEnumerate}

The tracked particle will be\+:
\begin{DoxyItemize}
\item Followed by the camera (view\+\_\+center = particle position)
\item Highlighted with info display (ID, position, velocity)
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Called on mouse click event
