\doxysubsection{Detailed Description}
N-\/body simulation of planetary system with 100k test particles. 

Simulates a central star with planets and a debris disk. Uses Barnes-\/\+Hut algorithm with quadtree spatial partitioning for efficient gravity calculation.

System setup\+:
\begin{DoxyItemize}
\item 1 central star (mass = \doxylink{classQuadTree_a32ecd5218cb8f164aaaf0eaded7064c9}{1.\+0})
\item 5 planets in circular orbits
\item 100,000 massless test particles (debris disk)
\end{DoxyItemize}

Integration\+: Hermite 4th order (default), configurable via TRANSPORT\+\_\+\+TYPE Timestep\+: \doxylink{classQuadTree_a32ecd5218cb8f164aaaf0eaded7064c9}{0.\+01} time units Visualization\+: SFML-\/based real-\/time rendering
