\doxysubsection{Detailed Description}
Implementation of SFML-\/based rendering for N-\/body simulation. 

Provides real-\/time visualization with\+:
\begin{DoxyItemize}
\item Velocity-\/based particle coloring (bound vs unbound orbits)
\item Interactive particle tracking
\item \doxylink{classQuadTree}{Quad\+Tree} visualization
\item Zoom and pan controls
\item Performance-\/optimized rendering (vertex arrays for 100k+ particles) 
\end{DoxyItemize}
