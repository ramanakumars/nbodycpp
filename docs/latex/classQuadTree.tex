\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Quad\+Tree$<$ T $>$}
Hierarchical spatial partitioning tree for 2D N-\/body simulation. 


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & \doxylink{classParticle}{Particle} type (must have position, mass, velocity members)\\
\hline
\end{DoxyTemplParams}
The \doxylink{classQuadTree}{Quad\+Tree} recursively subdivides 2D space into quadrants, storing particles in leaf nodes. This enables\+:
\begin{DoxyItemize}
\item Fast spatial queries (collision detection, neighbor search)
\item Barnes-\/\+Hut multipole approximation for gravity
\item O(\+N log N) force calculation instead of O(N²)
\end{DoxyItemize}

Tree structure\+:
\begin{DoxyItemize}
\item Each node covers a rectangular region (bounds)
\item Leaf nodes store up to MAX\+\_\+\+CAPACITY particles
\item Internal nodes have 4 children (quadrants\+: NW, NE, SW, SE)
\item Subdivision stops at MAX\+\_\+\+DEPTH
\end{DoxyItemize}

Barnes-\/\+Hut properties\+:
\begin{DoxyItemize}
\item Each node stores total mass and center of mass
\item Distant groups of particles treated as single mass
\item Opening angle criterion\+: s/d \texorpdfstring{$<$}{<} θ 
\end{DoxyItemize}
